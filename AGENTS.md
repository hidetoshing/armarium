# AGENTS.md

このファイルは、このリポジトリで作業するコーディングエージェント向けの共通指針です。サブディレクトリに別の `AGENTS.md` がある場合、そのディレクトリ以下ではより近いファイルの指示を優先してください。

## プロジェクト概要

Armarium は、ローカルイントラネット向けの軽量な OPDS 電子書籍サーバーです。

- 言語: Go 1.26
- 実行形態: 単一 Docker コンテナ
- HTTP 実装: Go 標準ライブラリ
- 対応形式: PDF、EPUB、CBZ、画像ファイルを収録した ZIP
- 認証: HTTP Basic 認証
- 永続データ: JSON 形式のメタデータキャッシュ

外部依存は、明確な必要性がない限り追加しないでください。軽量性、運用の単純さ、ローカル環境での扱いやすさを優先します。

## ディレクトリ構成

- `cmd/armarium/`: 実行エントリーポイント
- `internal/server/`: 設定、HTTP ハンドラー、認証、メタデータ処理
- `config.example.json`: 配布用の設定例
- `compose.yaml`: ローカル運用向け Docker Compose 構成
- `Dockerfile`: テスト、ビルド、実行イメージの定義

## 開発ルール

- コード、コメント、ドキュメント、コミットメッセージは原則として日本語で記述します。ただし、公開 API 名、プロトコル用語、識別子は一般的な英語表記を使用します。
- Go コードは `gofmt` を適用してください。
- エラーには処理対象が分かる文脈を付けてください。認証情報やファイル内容などの機密情報は含めないでください。
- ライブラリのパスは信頼できない入力として扱い、ディレクトリトラバーサルとシンボリックリンクによるライブラリ外への脱出を防いでください。
- HTTP Basic 認証が通信を暗号化しない点を変更やドキュメントで誤解させないでください。
- OPDS の URL、MIME タイプ、Atom XML の名前空間を変更する場合は、ハンドラー、テスト、README、`docs/opds-api.md` を同時に更新してください。
- システム構成、責務分担、設定、認証、メタデータ、キャッシュ、セキュリティ、Docker構成などの設計を変更する場合は、同じ変更内で `docs/architecture.md` と関連する設計ドキュメントを更新してください。実装とドキュメントが一致しない状態で作業を完了しないでください。
- `config.json`、電子書籍、キャッシュ、秘密鍵、認証情報をコミットしないでください。
- ユーザーの既存変更を無断で破棄したり、依頼範囲外のファイルを機械的に書き換えたりしないでください。

## テストと検証

変更内容に応じて、最低限次を実行してください。

```sh
gofmt -w cmd internal
go test ./...
git diff --check
```

ホストに Go がない場合は Docker を使用します。

```sh
docker compose build
```

`Dockerfile` のビルドステージは `go test ./...` を実行するため、コンテナビルドの成功によってテストとバイナリ生成の両方を確認できます。

認証、ファイルパス、ダウンロード処理を変更する場合は、正常系に加えて次の回帰テストを追加または維持してください。

- 未認証および不正な認証情報が `401 Unauthorized` になること
- `..` を使ったディレクトリトラバーサルを拒否すること
- ライブラリ外を指すシンボリックリンクを拒否すること
- 未対応形式を配信しないこと
- OPDS フィード内の取得 URL と MIME タイプが実際の配信結果と一致すること

## 設定と互換性

- 設定ファイルは環境変数 `ARMARIUM_CONFIG` で指定し、未指定時は `config.json` を使用します。
- 設定形式や既存エンドポイントの破壊的変更は避けてください。必要な場合は README、設定例、`docs/` の関連文書に移行方法を記載します。
- 新しい書籍形式を追加する場合は、拡張子判定、MIME タイプ、メタデータ抽出、ダウンロード、テスト、README、`docs/architecture.md`、`docs/opds-api.md` を一式で更新してください。

- `gofmt` とテストが成功している
- README、設定例、`docs/` の設計ドキュメントが実装と一致している
- 秘密情報、キャッシュ、電子書籍、生成物が差分に含まれていない
